# FGL_YOUTH

FGL Youth Development platform starter built with SvelteKit, PocketBase, Railway, Netlify, and GitHub Codespaces.

## Description

FGL_YOUTH is the starter application for the FGL Youth Development System and FLIHub platform. It is designed to support youth disc golf program management, participant profiles, parent oversight, program director administration, badges, certifications, season tracking, statistics, curriculum delivery, and long-term player development records.

## Deployment Shape

- **Frontend:** SvelteKit deployed to Netlify
- **Backend:** PocketBase deployed to Railway using Docker
- **Repository:** GitHub repo named `FGL_YOUTH`
- **Development:** GitHub Codespaces

Read `INSTALL.md` for the full setup guide.
